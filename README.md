# Secure File Vault UI

Frontend file vault concept with lock and unlock logic.

## Features
- Password-protected vault
- File metadata storage
- Lock / unlock state handling
- No backend required

## Usage
Open `index.html`, unlock the vault, and add files.

## Note
This is a UI and logic demo. Files are not actually encrypted.

## Purpose
Demonstrates secure UI flows, file handling, and protected state concepts.
